## DiscoverService

```ts
import { DiscoverService } from 'jiosaavn-sdk'

const discoverService = new DiscoverService()
```

---

### getTopArtists

Fetch the most popular artists on JioSaavn.

```ts
const artists = await discoverService.getTopArtists()
```

#### Parameters

None.

#### Return Type — TopArtistModel[]

<TypeTable
  type={{
    id: {
      description: 'Artist ID',
      type: 'string',
      required: true,
    },
    name: {
      description: 'Artist name',
      type: 'string',
      required: true,
    },
    image: {
      description: 'Artist image URL',
      type: 'string',
      required: true,
    },
    followerCount: {
      description: 'Number of followers',
      type: 'number',
      required: true,
    },
    isFollowed: {
      description: 'Whether the current user follows this artist',
      type: 'boolean',
      required: true,
    },
    url: {
      description: 'JioSaavn permalink',
      type: 'string',
      required: true,
    },
  }}
/>

### getFeaturedPlaylists

Fetch editorially curated playlists. Supports pagination.

```ts

  page: 1,
  limit: 10
})
```
