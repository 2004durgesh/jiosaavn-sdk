The **JioSaavn SDK** is a universal, isomorphic TypeScript SDK that lets you interact with the JioSaavn music platform. Search for songs, albums, artists, and playlists, fetch detailed metadata, and generate download links — all from a clean, type-safe API.

## Features

- **Type-safe** — Full TypeScript support with Zod-validated models
- **Isomorphic** — Works in Node.js, Deno, Bun, and browser environments
- **Zero config** — No API keys or authentication required
- **Modular** — Use only the services you need
