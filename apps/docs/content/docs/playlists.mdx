## PlaylistService

```ts
import { PlaylistService } from 'jiosaavn-sdk'

const playlistService = new PlaylistService()
```

---

### getPlaylistById

Fetch a playlist by its ID with pagination support.

```ts
const playlist = await playlistService.getPlaylistById({
  id: '159470188',
  limit: 20,
  page: 0
})
```

#### Parameters

<TypeTable
  type={{
    id: {
      description: 'The unique playlist ID',
      type: 'string',
      required: true,
    },
    limit: {
      description: 'Max number of songs to include',
      type: 'number',
      required: true,
    },
    page: {
      description: 'Page number for pagination',
      type: 'number',
      required: true,
    },
  }}
/>

**Returns:** `Promise<PlaylistModel>`

**Throws:** `SaavnError(404)` if the playlist is not found.

---

### getPlaylistByLink

Fetch a playlist by a JioSaavn link token.

```ts
const playlist = await playlistService.getPlaylistByLink({
  token: 'xY3bZ9qR',
  limit: 20,
  page: 0
})
```

#### Parameters

<TypeTable
  type={{
    token: {
      description: 'Token extracted from a JioSaavn playlist URL',
      type: 'string',
      required: true,
    },
    limit: {
      description: 'Max number of songs to include',
      type: 'number',
      required: true,
    },
    page: {
      description: 'Page number for pagination',
      type: 'number',
      required: true,
    },
  }}
/>

**Returns:** `Promise<PlaylistModel>`

**Throws:** `SaavnError(404)` if the playlist is not found.
