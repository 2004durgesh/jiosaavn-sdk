## SaavnError

All services throw a `SaavnError` when an operation fails. This is a standard `Error` subclass with an additional `statusCode` property.

```ts
import { SaavnError } from 'jiosaavn-sdk'
```

### Common Error Codes

| Code | Meaning | Example |
|------|---------|---------|
| `404` | Resource not found | Song ID doesn't exist |
| `500` | Internal error | Failed to create song station |

### Catching Errors

```ts
import { SongService, SaavnError } from 'jiosaavn-sdk'

const songService = new SongService()

try {
  await songService.getSongByIds({ songIds: 'nonexistent' })
} catch (error) {
  if (error instanceof SaavnError) {
    switch (error.statusCode) {
      case 404:
        console.log('Song not found:', error.message)
        break
      default:
        console.log('Unexpected error:', error.message)
    }
  }
}
```
