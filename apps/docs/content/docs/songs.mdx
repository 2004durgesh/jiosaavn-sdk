## SongService

```ts
import { SongService } from 'jiosaavn-sdk'

const songService = new SongService()
```

---

### getSongByIds

Fetch one or more songs by their IDs.

```ts
const songs = await songService.getSongByIds({ songIds: '3IoDK8qI' })
```

#### Parameters

<TypeTable
  type={{
    songIds: {
      description: 'Comma-separated list of song IDs',
      type: 'string',
      required: true,
    },
  }}
/>

**Returns:** `Promise<SongModel[]>`

**Throws:** `SaavnError(404)` if no songs are found.

---

### getSongByLink

Fetch songs by a JioSaavn link token.

```ts
const songs = await songService.getSongByLink('OgwhbhtDRwM')
```

#### Parameters

<TypeTable
  type={{
    token: {
      description: 'The token extracted from a JioSaavn song URL',
      type: 'string',
      required: true,
    },
  }}
/>

**Returns:** `Promise<SongModel[]>`

**Throws:** `SaavnError(404)` if no songs are found.

---

### getSongSuggestions

Get song suggestions based on a given song.

```ts
const suggestions = await songService.getSongSuggestions({
  songId: 'yDeAS8Eh',
  limit: 10
})
```

#### Parameters

<TypeTable
  type={{
    songId: {
      description: 'The song ID to base suggestions on',
      type: 'string',
      required: true,
    },
    limit: {
      description: 'Max number of suggestions to return',
      type: 'number',
      required: true,
    },
  }}
/>

**Returns:** `Promise<SongModel[]>`

**Throws:** `SaavnError(404)` if no suggestions are found.

---

### createSongStation

Create a radio station from a song ID.

```ts
const stationId = await songService.createSongStation('3IoDK8qI')
```

#### Parameters

<TypeTable
  type={{
    songIds: {
      description: 'Song ID to create the station from',
      type: 'string',
      required: true,
    },
  }}
/>

**Returns:** `Promise<string>` — The station ID.

**Throws:** `SaavnError(500)` if the station could not be created.
