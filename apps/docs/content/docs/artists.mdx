## ArtistService

```ts
import { ArtistService } from 'jiosaavn-sdk'

const artistService = new ArtistService()
```

---

### getArtistById

Fetch an artist profile by ID with their top songs and albums.

```ts
const artist = await artistService.getArtistById({
  artistId: '1274170',
  page: 0,
  songCount: 10,
  albumCount: 10,
  sortBy: 'popularity',
  sortOrder: 'asc'
})
```

#### Parameters

<TypeTable
  type={{
    artistId: {
      description: 'The unique artist ID',
      type: 'string',
      required: true,
    },
    page: {
      description: 'Page number for paginated results',
      type: 'number',
      required: true,
    },
    songCount: {
      description: 'Number of top songs to include',
      type: 'number',
      required: true,
    },
    albumCount: {
      description: 'Number of top albums to include',
      type: 'number',
      required: true,
    },
    sortBy: {
      description: 'Sort criteria for results',
      type: "'popularity' | 'latest' | 'alphabetical'",
      required: true,
    },
    sortOrder: {
      description: 'Sort direction',
      type: "'asc' | 'desc'",
      required: true,
    },
  }}
/>

**Returns:** `Promise<ArtistModel>`

**Throws:** `SaavnError(404)` if the artist is not found.

### getArtistByLink

Fetch an artist by a JioSaavn link token. Same parameters as `getArtistById`, but with `token` instead of `artistId`.

```ts
const artist = await artistService.getArtistByLink({
  token: 'r-OWIKgpX2I_',
  page: 0,
  songCount: 10,
  albumCount: 10,
  sortBy: 'popularity',
  sortOrder: 'asc'
})
```
